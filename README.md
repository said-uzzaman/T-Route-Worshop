# T-Route Workshop

Materials for the **T-Route: Channel Routing in the National Water Model** hands-on training workshop. This repository contains the presentations, agenda, example datasets, and guidance documents used throughout the sessions.

T-Route is a tree-based, one-dimensional channel routing model that operates on vector-based river networks and serves as the primary channel routing module in the NOAA Next Generation Water Resources Modeling (NextGen) Framework, supporting both Muskingum-Cunge (MC) and Diffusive Wave (DW) routing schemes.

## Repository Contents

| Folder | Description |
|---|---|
| `agenda/` | Workshop schedule and session outline |
| `presentations/` | Slide decks for each session |
| `data/` | Example and subset datasets (hydrofabric, forcing, USGS observations) |
| `docs/` | Setup instructions and additional guidance |

## Getting Started

```bash
git clone https://github.com/said-uzzaman/T-Route-Worshop.git
cd T-Route-Worshop
```

Follow the setup instructions in `docs/` to build the environment, then work through the materials in each folder in session order.
